üìÅ What is File Handling?

File handling is the process of working with files stored on your computer's storage system. In Python, we can:

  • Create new files
  • Read data from existing files
  • Write data to files
  • Modify existing file content
  • Delete files when needed

Python provides built-in functions and methods to perform these operations efficiently and safely.

üîì The open() Function

The open() function is your gateway to file operations. It creates a file object that you can use to read from or write to a file.

Syntax:

file_object = open("filename", "mode")

Example:

# Opening a file in read mode file = open("example.txt", "r") # Opening a file in write mode file = open("output.txt", "w")

üéõÔ∏è File Modes

File modes determine how you can interact with the file. Here are the most common modes:

Mode Description File Pointer Creates File?
r Read only (default) Beginning No
w Write only (overwrites existing) Beginning Yes
a Append only End Yes
r+ Read and Write Beginning No
w+ Write and Read (overwrites existing) Beginning Yes
a+ Append and Read End Yes

üìñ Reading Files

Python provides several methods to read file content:

1. read() - Read entire file

# Read entire file content file = open("sample.txt", "r") content = file.read() print(content) file.close()

2. readline() - Read one line

# Read one line at a time file = open("sample.txt", "r") first_line = file.readline() print(first_line) file.close()

3. readlines() - Read all lines into a list

# Read all lines into a list file = open("sample.txt", "r") lines = file.readlines() for line in lines: print(line.strip()) # strip() removes newline characters file.close()

‚úçÔ∏è Writing Files

Use the write() method to add content to files:

1. Basic Writing

# Write to a file (overwrites existing content) file = open("output.txt", "w") file.write("Hello, World!\n") file.write("This is a new line.") file.close()

2. Appending to Files

# Append to existing file file = open("output.txt", "a") file.write("\nThis line is appended.") file.close()

3. Writing Multiple Lines

# Write list of lines lines = ["Line 1\n", "Line 2\n", "Line 3\n"] file = open("multi_line.txt", "w") file.writelines(lines) file.close()

üîí Closing Files with close()

Always close files after use to free up system resources and ensure data is properly saved.

# Manual closing file = open("example.txt", "r") content = file.read() file.close() # Important: Always close the file!

‚ö†Ô∏è Why Closing Files is Important:

  • Frees up system memory and resources
  • Ensures all data is written to the file
  • Prevents file corruption
  • Allows other programs to access the file
  • Avoids "too many open files" errors

üõ°Ô∏è The 'with' Statement (Best Practice)

The with statement automatically handles file closing, even if an error occurs:

Reading with 'with':

# Automatic file closing with 'with' statement with open("example.txt", "r") as file: content = file.read() print(content) # File is automatically closed here!

Writing with 'with':

# Writing with automatic file management with open("output.txt", "w") as file: file.write("Hello, World!") file.write("\nPython File Handling") # File is automatically closed and saved!

üí° Best Practices Summary

üåü Follow These Guidelines:

  • Always use the 'with' statement for automatic file management
  • Handle exceptions with try-except blocks
  • Use appropriate file modes for your needs
  • Close files explicitly if not using 'with' statement
  • Use strip() to remove unwanted whitespace when reading
  • Check if files exist before performing operations
  • Use meaningful variable names for file objects
  • Be mindful of file paths and working directories
