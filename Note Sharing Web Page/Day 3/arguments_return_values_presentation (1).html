Arguments and Return Values

Understanding Function Communication

How functions receive input and send output

🔄 Input → Function → Output

What are Arguments?

Arguments are values passed to a function when it's called. They provide input data for the function to work with.

Key Points:

  • Arguments are the actual values sent to a function
  • Parameters are the variables that receive these values
  • Functions can have zero, one, or multiple arguments
  • Arguments are passed in a specific order

Simple Example:

def greet(name):        # 'name' is a parameter
    print(f"Hello, {name}!")

greet("Alice")          # "Alice" is an argument

Types of Arguments

1. Positional Arguments

Arguments passed in a specific order

def add(a, b):
    return a + b

result = add(5, 3)      # 5 goes to 'a', 3 goes to 'b'
print(result)           # Output: 8

2. Keyword Arguments

Arguments passed with parameter names

def introduce(name, age, city):
    print(f"I'm {name}, {age} years old from {city}")

introduce(age=25, name="Bob", city="New York")
# Order doesn't matter with keyword arguments

Default Arguments

Parameters can have default values, making them optional when calling the function.

def create_profile(name, age=18, country="Unknown"):
    return f"Name: {name}, Age: {age}, Country: {country}"

# Different ways to call:
print(create_profile("Alice"))
# Output: Name: Alice, Age: 18, Country: Unknown

print(create_profile("Bob", 25))
# Output: Name: Bob, Age: 25, Country: Unknown

print(create_profile("Charlie", 30, "USA"))
# Output: Name: Charlie, Age: 30, Country: USA

Note: Default parameters must come after non-default parameters!

Variable Arguments

*args - Variable Positional Arguments

def sum_all(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(sum_all(1, 2, 3))           # Output: 6
print(sum_all(1, 2, 3, 4, 5))    # Output: 15

**kwargs - Variable Keyword Arguments

def print_info(**details):
    for key, value in details.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30, job="Engineer")
# Output:
# name: Alice
# age: 30
# job: Engineer

Return Values

Return values are the output that functions send back to the caller.

Basic Return:

def square(x):
    return x * x

result = square(4)      # Function returns 16
print(result)           # Output: 16

Multiple Return Values:

def divide_and_remainder(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder

q, r = divide_and_remainder(17, 5)
print(f"Quotient: {q}, Remainder: {r}")
# Output: Quotient: 3, Remainder: 2

Practical Examples

Example 1: Calculator Function

def calculator(operation, a, b):
    if operation == "add":
        return a + b
    elif operation == "subtract":
        return a - b
    elif operation == "multiply":
        return a * b
    elif operation == "divide":
        return a / b if b != 0 else "Cannot divide by zero"

print(calculator("add", 10, 5))        # Output: 15
print(calculator("multiply", 3, 4))    # Output: 12

Example 2: Student Grade System

def calculate_grade(*scores, bonus=0):
    if not scores:
        return "No scores provided"
    
    average = sum(scores) / len(scores) + bonus
    
    if average >= 90:
        return "A"
    elif average >= 80:
        return "B"
    elif average >= 70:
        return "C"
    else:
        return "F"

grade = calculate_grade(85, 92, 78, bonus=2)
print(f"Grade: {grade}")    # Output: Grade: A

Best Practices

✅ Do:

  • Use descriptive parameter names
  • Provide default values for optional parameters
  • Document your function's expected arguments and return values
  • Keep functions focused on a single task
  • Always return consistent data types

❌ Avoid:

  • Too many parameters (more than 5-7)
  • Modifying mutable default arguments
  • Functions that sometimes return values and sometimes don't
  • Using unclear or abbreviated parameter names

Good Example:

def format_currency(amount, currency_code="USD", decimal_places=2):
    """
    Format a number as currency.
    
    Args:
        amount: The monetary amount
        currency_code: Currency symbol (default: USD)
        decimal_places: Number of decimal places (default: 2)
    
    Returns:
        Formatted currency string
    """
    return f"{currency_code} {amount:.{decimal_places}f}"
