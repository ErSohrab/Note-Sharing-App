What are Loops? üîÑ

Loops are control structures that allow you to repeat a block of code multiple times based on certain conditions.

Why Use Loops?

  • Efficiency: Avoid writing repetitive code
  • Automation: Process large datasets automatically
  • Dynamic Execution: Handle unknown quantities of data
  • Clean Code: Make programs more readable and maintainable

Without Loops (Repetitive):

print("Hello") print("Hello") print("Hello") print("Hello") print("Hello")

With Loops (Efficient):

for i in range(5): print("Hello")

For Loops - The Basics üîÑ

Syntax Structure:

for variable in iterable: # Code block to execute statement1 statement2

Example 1: Range Function

# Print numbers 0 to 4 for i in range(5): print(i) # Output: 0, 1, 2, 3, 4

Example 2: List Iteration

fruits = ["apple", "banana", "orange"] for fruit in fruits: print(f"I like {fruit}")

Example 3: String Iteration

for char in "Python": print(char) # Output: P, y, t, h, o, n

Example 4: Dictionary Iteration

student = {"name": "Alice", "age": 20} for key, value in student.items(): print(f"{key}: {value}")
üí° Pro Tip: For loops in Python are more versatile than in many other languages - they can iterate over any iterable object!

For Loops - Advanced Techniques üöÄ

enumerate() Function

colors = ["red", "green", "blue"] for index, color in enumerate(colors): print(f"{index}: {color}") # Output: # 0: red # 1: green # 2: blue

zip() Function

names = ["Alice", "Bob", "Charlie"] ages = [25, 30, 35] for name, age in zip(names, ages): print(f"{name} is {age} years old")

Range with Parameters

# range(start, stop, step) for i in range(2, 10, 2): print(i) # Output: 2, 4, 6, 8 # Reverse iteration for i in range(5, 0, -1): print(i) # Output: 5, 4, 3, 2, 1

Nested For Loops

for i in range(3): for j in range(3): print(f"({i}, {j})", end=" ") print() # New line

üéØ Best Practices for For Loops:

  • Use descriptive variable names (not just 'i', 'j')
  • Prefer enumerate() when you need both index and value
  • Use zip() to iterate over multiple sequences simultaneously
  • Avoid modifying the sequence you're iterating over

While Loops - The Basics üîÅ

Syntax Structure:

while condition: # Code block to execute statement1 statement2 # Don't forget to update the condition!

Example 1: Basic Counter

count = 0 while count < 5: print(f"Count: {count}") count += 1 # Output: Count: 0, 1, 2, 3, 4

Example 2: User Input

password = "" while password != "secret": password = input("Enter password: ") if password != "secret": print("Wrong password!") print("Access granted!")

Example 3: List Processing

numbers = [1, 2, 3, 4, 5] while numbers: num = numbers.pop() print(f"Processing: {num}") # Processes in reverse order

Example 4: Mathematical Calculation

# Calculate factorial n = 5 factorial = 1 original_n = n while n > 0: factorial *= n n -= 1 print(f"{original_n}! = {factorial}")
‚ö†Ô∏è Warning: Always ensure your while loop condition eventually becomes False to avoid infinite loops!

While vs For Loops - When to Use What? ü§î

‚úÖ Use FOR loops when:

  • You know the number of iterations
  • Iterating over collections (lists, strings, dicts)
  • Working with ranges of numbers
  • Processing each item in a sequence

Perfect For Scenarios:

# Processing a list for item in shopping_list: print(item) # Repeating an action N times for i in range(10): print("Hello World")

‚úÖ Use WHILE loops when:

  • Condition-based iteration
  • Unknown number of iterations
  • User input validation
  • Game loops or real-time processing

Perfect While Scenarios:

# User input until valid while not valid_input: user_input = input("Enter: ") valid_input = validate(user_input) # Game loop while game_running: handle_events() update_game() render()

üéØ Decision Matrix:

Scenario Recommended Loop Reason
Process each file in a folder FOR Known collection to iterate
Wait for user to guess correctly WHILE Unknown number of attempts
Print multiplication table FOR Fixed range (1-10)
Read data until EOF WHILE Condition-based termination

Loop Control: Break Statement üõë

The break statement immediately terminates the loop and transfers control to the statement after the loop.

Break in For Loop

# Find first even number numbers = [1, 3, 7, 8, 9] for num in numbers: if num % 2 == 0: print(f"Found even number: {num}") break print(f"Checking: {num}") # Output: # Checking: 1 # Checking: 3 # Checking: 7 # Found even number: 8

Search Algorithm Example

# Linear search target = "apple" fruits = ["banana", "apple", "orange"] found = False for i, fruit in enumerate(fruits): if fruit == target: print(f"Found {target} at index {i}") found = True
